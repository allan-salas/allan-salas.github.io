BAC Credomatic App

BAC Credomatic is one of the main banks in the region, which has a web platform and an application.

It also has its own Design System to standardize the different platforms and channels of contact with the user.


  • What i done in this project?

  • User flows
  • Wireframes
  • Journey maps
  • Conduct workshops to define projects
  • Collaborate with the creation of the Design System
  • Design features for app and web banking platform such as "Withdrawal without card"
  • User testing with prototypes
  • Some app animations
